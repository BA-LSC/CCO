# CCO Mobile (Expo)

This app is **not** part of the root Bun workspace so Expo dependencies do not inflate web/API installs.

## Setup

From the repository root:

```bash
cd apps/mobile
bun install
```

## Run

```bash
bun run dev
# or
bun run ios
bun run android
```

Configure API URLs in the app to point at your local or production CCO API.
